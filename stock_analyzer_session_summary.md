# A股消息驱动分析助手 项目开发 Session 总结

**日期**：2026年7月  
**开发者**：堂铭 + Grok 协作开发  
**仓库**：https://github.com/wangtangming830411-star/stock_analyzer

## 1. 项目背景与目标

用户希望开发一个**以消息驱动为核心**的A股股票分析工具，重点解决传统K线软件的痛点：
- 优先抓取**第一手时事政治经济消息**（政策、地缘、产业新闻）
- 自动分析消息对**行业/板块**的影响（利好/利空）
- 推荐相关行业代表性股票（5-10只）
- 传统桌面窗口风格，清晰一目了然，便于迅速决策

## 2. 已完成的功能（成功部分）

### 核心框架
- Python + CustomTkinter 传统桌面 GUI（暗色主题、可拖动分割）
- Tab 切换：新闻 / 影响推荐 / 我的持仓
- Git 初始化 + 版本控制 + 上传 GitHub

### 数据获取
- akshare 真实新闻获取（东方财富来源，已修复参数问题）
- 多股票代码并行获取新闻（58+条真实新闻）
- 模拟数据作为备用

### 新闻处理与智能分析
- `NewsProcessor`：自动生成总结 + 关键词匹配行业影响
- `industry_mapping.json`：包含军工、AI、人形机器人、固态电池、新能源、半导体、低空经济等2026年热点行业
- 影响标签显示（利好标识）

### 推荐系统
- `IndustryMapper`：根据新闻提及频率排序行业
- 为每个影响行业推荐5-10只代表性股票（含模拟真实数据）

### 其他功能
- 本地 SQLite 缓存（保存总结 + 链接）
- 手动刷新 + 后台自动刷新（30分钟）
- 持仓 Tab（手动添加代码测试）
- 错误处理与多线程安全

### 开发过程亮点
- 完整走完 SDLC（需求→设计→编码→测试→迭代）
- 多次修复 Tkinter、akshare 参数、线程安全、UI渲染等问题
- 项目结构清晰（data/、gui 逻辑分离）

## 3. 现存尚未解决的问题

1. **新闻内容显示问题**（当前主要阻塞）
   - 终端显示“获取到58条真实新闻”
   - 但 GUI 新闻 Tab 中只显示条数，具体标题/总结/影响标签不渲染

2. **推荐 Tab 内容为空**
   - 行业推荐和股票列表未正常显示

3. **持仓功能基础版**
   - 仅支持手动添加代码，无实时价格、盈亏计算、走势分析

4. **新闻来源** 
   - 目前主要依赖 akshare，可扩展 RSS、政府官网、龙头公司公告

5. **高级功能待实现**
   - 本地 Qwen LLM 智能总结与深度分析
   - 持仓实时数据更新 + 盈亏计算
   - 导出报告（Markdown/Word/PDF）
   - 打包成独立 Mac App
